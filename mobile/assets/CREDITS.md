# Asset credits

## s1000rr.m4a (launch splash engine sound)

User-supplied engine recording — a Yamaha R6 inline-4 "idle to redline" clip
generated with Adobe Firefly (provided by the project owner). Processed for the
splash: downmixed to mono, normalised to ~-2.5 dBFS, short fade-in/out, encoded
to AAC/m4a (~2 s).

Licensing follows Adobe Firefly's generated-content terms; keep the original
source file with the project if you need to re-export. To swap it, overwrite
this file (keep it short, ~2-3 s, m4a/mp3/wav).
